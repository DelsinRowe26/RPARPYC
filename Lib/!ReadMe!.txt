RPA Extractor for Windows

Compiled by Sn34kyMofo.com via the following resources:

Python 2.7.9: https://www.python.org/downloads/
RPATool by Shizmob: https://github.com/Shizmob/rpatool
py2exe: http://www.py2exe.org/

Инструкция по применению:

Распаковываем этот архив в отдельную папку.
Копируем в него нужный нам RPA файл.
Открываем в блокноте файл cmd.bat
Заменяем ИМЯ_ФАЙЛА на имя вашего файла.
Сохраняем.
Запускаем cmd.bat
Ждем, пока архив распакуется.
Все файлы с архива будут в одноименной папке.
